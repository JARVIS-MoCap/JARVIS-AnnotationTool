Build Chain Setup:
	-> get git (regular installer, but make sure it is listed after all the Strawberry perl stuff in the path variable)
	-> Install Strawberry Perl (32 Bit Version)
	-> Install MinGW, DONT get it from source forge but get the win-builds thingy from the mingw-something.org website
	-> Install CMake (just get the official version from the cmake website)

Build static QT5:
	-> Get QT from git with: git clone git://code.qt.io/qt/qt5.git
	-> checkout desired version with: git checkout 5.15.2 (this Version works, others had problems)
	-> run perl init-repository inside the rep cloned from git
	-> Make sure MinGW/bin and all the Strawberry perl stuff is in your path and run:
	   ../qt5/configure -static -release -optimize-size -no-pch -prefix C:/Qt5static_v5.15.2 -opensource -confirm-license -nomake tools -nomake tests -nomake examples -no-opengl -skip webengine -skip qt3d -skip qtandroidextras -skip qtcanvas3d -skip qtconnectivity -skip qtdatavis3d -skip qtdeclarative -skip qtdoc -skip qtgamepad -skip qtgraphicaleffects -skip qtlocation -skip qtmacextras -skip qtnetworkauth -skip qtpurchasing -skip qtquickcontrols -skip qtquickcontrols2 -skip qtscript -skip qtscxml -skip qtsensors -skip qtserialbus -skip qtserialport -skip qtquick3d -skip qtspeech -skip qttools -skip qtsensors -skip qttranslations -skip qtvirtualkeyboard -skip qtwayland -skip qtwebchannel -skip qtwebengine -skip qtwebsockets -skip qtwebview -skip qtwinextras -skip qtxmlpatterns -skip qtserialport -skip qtserialport -skip qtwebglplugin
	-> run mingw32-make -jx (with x being the number of threads on your system e.g. -j12)
	-> run mingw32-make install

Building static OpenCV:
	-> get opencv from git with: git clone https://github.com/opencv/opencv.git
	-> git checkout 4.5.1 (DONT use 4.5.3, somehow it doesn't build libade.a)
	-> configure opencv with:
	   cmake -DOPENCV_ENABLE_ALLOCATOR_STATS=OFF -DCMAKE_BUILD_TYPE=RELEASE -DBUILD_TIFF=OFF -DBUILD_JPEG=OFF -DBUILD_ZLIB=OFF -DBUILD_WEBG=OFF -DBUILD_PNG=OFF -DBUILD_WEBG=OFF -DWITH_OPENEXR=OFF -DWITH_OPENJPEG=OFF -DWITH_JASPER=OFF -DWITH_PROTOBUF=OFF -DWITH_QUIRC=OFF -DBUILD_SHARED_LIBS=OFF -DBUILD_LIST="core,calib3d,imgproc,videoio" -DCMAKE_INSTALL_PREFIX=C:opencv/opencv_static ../opencv -G "MinGW Makefiles"	 
	-> run cmake --build . -jx (with x being the number of threads on your system e.g. -j12)
	-> run cmake --install .


Notes:
   -> I had some weird conflict coming from Qt referencing the perland Cmake referenncing the MinGW dlls. Make sure they both build using the same system
