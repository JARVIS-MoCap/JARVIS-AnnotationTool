Build static QT5:
	-> Get QT from git with: git clone git://code.qt.io/qt/qt5.git
	-> checkout desired version with: git checkout 5.15.2 (this Version works, others had problems)
	-> configure with:
	./qt5/configure -static -release -optimize-size -no-pch -prefix /home/timo/Qt_build/Qt5static_v5.15.2 -opensource -confirm-license -nomake tools -nomake tests -nomake examples -no-opengl -skip webengine -skip qt3d -skip qtandroidextras -skip qtcanvas3d -skip qtconnectivity -skip qtdatavis3d -skip qtdeclarative -skip qtdoc -skip qtgamepad -skip qtgraphicaleffects -skip qtlocation -skip qtmacextras -skip qtnetworkauth -skip qtpurchasing -skip qtquickcontrols -skip qtquickcontrols2 -skip qtscript -skip qtscxml -skip qtsensors -skip qtserialbus -skip qtserialport -skip qtquick3d -skip qtspeech -skip qttools -skip qtsensors -skip qttranslations -skip qtvirtualkeyboard -skip qtwayland -skip qtwebchannel -skip qtwebengine -skip qtwebsockets -skip qtwebview -skip qtwinextras -skip qtxmlpatterns -skip qtserialport -skip qtserialport -skip qtwebglplugin


Building static FFMPEG:
	-> get from git: https://github.com/zimbatm/ffmpeg-static
	->replace flags in "build.sh" with this:
		--pkg-config-flags="--static" \
		--extra-cflags="-I$TARGET_DIR/include" \
		--extra-ldflags="-L$TARGET_DIR/lib" \
		--extra-libs="-lpthread -lm -lz" \
		--extra-ldexeflags="-static" \
		--bindir="$BIN_DIR" \
		--enable-pic \
		--enable-fontconfig \
		--enable-gpl \
		--enable-version3 \
		--enable-libfreetype \
		--enable-libmp3lame \
		--enable-libopenjpeg \
		--enable-libopus \
		--enable-librtmp \
		--enable-libsoxr \
		--enable-libspeex \
		--enable-libtheora \
		--enable-libvorbis \
		--enable-libvpx \
		--enable-libwebp \
		--enable-libx264 \
		--enable-libx265 \
		--enable-libxvid \
		--enable-libzimg \
		--enable-avresample
	-> use the ./build-ubuntu.sh script!


Building static OpenCV:
	-> get opencv from git with: git clone https://github.com/opencv/opencv.git
	-> git checkout 4.5.1 (DONT use 4.5.3, somehow it doesn't build libade.a)
	-> make sure PKG_CONFIG_PATH is set to point to static FFMPEG directory
	-> configure opencv with:
	cmake -DOPENCV_ENABLE_ALLOCATOR_STATS=OFF -DCMAKE_BUILD_TYPE=RELEASE -DBUILD_TIFF=OFF -DWITH_TIFF=OFF -DBUILD_JPEG=OFF -DBUILD_ZLIB=ON -DBUILD_WEBP=ON -DBUILD_PNG=OFF -DWITH_OPENEXR=OFF -DWITH_OPENJPEG=OFF -DWITH_JASPER=OFF -DWITH_PROTOBUF=OFF -DWITH_QUIRC=OFF -DWITH_1394=OFF -DWITH_V4L=OFF  -DWITH_GSTREAMER=ON -DWITH_FFMPEG=ON -DWITH_GTK=OFF -DBUILD_SHARED_LIBS=OFF D OPENCV_EXTRA_EXE_LINKER_FLAGS="-Wl,-rpath,/home/timo/ffmpeg_static/ffmpeg-static/target/lib" -DBUILD_LIST="core,calib3d,imgproc,videoio" -DCMAKE_INSTALL_PREFIX=/home/timo/OpenCV/opencv_static3 ../opencv-master
	-> run make -jx (with x being the number of threads on your system e.g. -j12)
	-> run make install .
