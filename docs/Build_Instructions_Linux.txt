Build static QT5:
	-> Get QT from git with: git clone git://code.qt.io/qt/qt5.git
	-> checkout desired version with: git checkout 5.15.2 (this Version works, others had problems)
	-> inside qt5 run./init-repository
	-> configure with:
	../qt5/configure -release -no-pch -prefix /home/timo/Qt_build/Qt5shared_v5.15.2 -opensource -confirm-license -nomake tools -nomake tests -nomake examples -no-opengl -skip webengine -skip qt3d -skip qtandroidextras -skip qtcanvas3d -skip qtconnectivity -skip qtdatavis3d -skip qtdeclarative -skip qtdoc -skip qtgamepad -skip qtgraphicaleffects -skip qtlocation -skip qtmacextras -skip qtnetworkauth -skip qtpurchasing -skip qtquickcontrols -skip qtquickcontrols2 -skip qtscript -skip qtscxml -skip qtsensors -skip qtserialbus -skip qtquick3d -skip qtspeech -skip qttools -skip qtsensors -skip qttranslations -skip qtvirtualkeyboard -skip qtwayland -skip qtwebchannel -skip qtwebengine -skip qtwebsockets -skip qtwebview -skip qtwinextras -skip qtxmlpatterns  -skip qtwebglplugin -no-openssl -no-vulkan

Building static OpenCV:
	-> get opencv from git with: git clone https://github.com/opencv/opencv.git
	-> git checkout 4.5.1 (DONT use 4.5.3, somehow it doesn't build libade.a)
	-> make sure PKG_CONFIG_PATH is set to point to static FFMPEG directory
	-> configure opencv with:
	cmake -DOPENCV_ENABLE_ALLOCATOR_STATS=OFF -DCMAKE_BUILD_TYPE=RELEASE -DBUILD_TIFF=OFF -DWITH_TIFF=OFF -DBUILD_JPEG=OFF -DBUILD_ZLIB=OFF -DBUILD_WEBP=OFF -DBUILD_PNG=OFF -DWITH_OPENEXR=OFF -DWITH_OPENJPEG=OFF -DWITH_JASPER=OFF -DWITH_PROTOBUF=OFF -DWITH_QUIRC=OFF -DWITH_1394=OFF -DWITH_V4L=OFF  -DWITH_GSTREAMER=ON -DWITH_FFMPEG=ON -DWITH_GTK=OFF -DBUILD_SHARED_LIBS=ON -DBUILD_LIST="core,calib3d,imgproc,videoio,aruco" -DCMAKE_INSTALL_PREFIX=/home/trackingsetup/opencv/opencv_shared -DOPENCV_EXTRA_MODULES_PATH=../opencv_contrib-master/modules ../opencv-master

	-> run make -jx (with x being the number of threads on your system e.g. -j12)
	-> run make install .
