Create Calibration

In this section we assume that you have already successfully recorded calibration videos as described in Recording Calibration Videos.

Folder Structure

To use your recorded calibration videos with the calibration tool you first need to arrange them according to the following naming scheme and folder structure. Below you can see an example layout for a 3 camera setup that useses Camera_1 as the primary camera.

Note that if you recorded the extrinsic videos for all cameras in one recording you can still use the structure as shown above by copying the video file of your primary camera to all the subfolders.

Detailed description of all the Settings

General

  • Calibration Set Name: Choose a descriptive name for your calibration set. This will be the name of the folder your calibration files will be stored in.
  • Calibration Set Savepath: Set the path to the directory your calibration set will be saved in.
  • Seperate Recordings for Intrinsics: Select Yes if you have recorded seperate intrinsic videos for each camera.
  • Calibrate Extrinsics: Select No if you want to calibrate only the intrinsic parameters of your cameras.
  • Intrinsics Folder Path: Path to the directory containing all your intrinsics recordings.
  • Extrinsic Folder Path: Path to the directory containing all your extrinsic recordings, structured as in the example above. If you recorded combined intrinsics and extrinsics videos you don't have to specify an intrinsics path.
  • Update Camera Names Button: Click this after selecting your folder paths to auto-fill the lists on the right. You can still edit them after running the auto-fill of course.

Calibration Parameters

  • Max. Sampling Frame Rate: Maximum frame rate your videos will be sampled at to extract calibration images. If this number is higher than the frame rate of your videos your recording frame rate will be used.
  • Frames for Intrinsics Calibration: Number of succesfully detected checkerboards to be used for intrinsics calibration. 20 Frames is a good starting point, add more if you want higher precision. More than 100 is not recommended.
  • Frames for Extrinsics Calibration: Number of succesfully detected checkerboards to be used for extrinsics calibration. 20 Frames is a good starting point, add more if you want higher precision. More than 100 is not recommended.
  • Save Debug Images: If Yes is selected a debug folder containing visualizations of all detected checkerboard corners is created within the calibration set directory.

Checkerboard Layout

  • Patter Width/Height: Pattern size of the checkerboard used for calibration. Compare your checkerboard to the visualization to make sure you're using the correct pattern size.
  • Side Length [mm]: Side length of a single square of your checkerboard in mm.
